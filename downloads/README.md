# 学习资源库原创下载包

这里保存“学习资源库”资源下载页提供的原创资料。每个目录是一套可独立使用的资料包，页面下载链接指向同名 ZIP 文件。

## 当前资料包

- `ai-tool-source-verification-kit`：AI 工具来源核验表，用于试用新工具前确认来源、权限、数据边界和验收结果。
- `open-source-project-release-kit`：开源项目发布检查包，用于补齐 README、许可证、运行步骤和维护记录。
- `website-launch-checklist`：静态网站上线检查表。
- `git-github-cheatsheet`：Git 与 GitHub 中文速查包。
- `vscode-starter-config`：VS Code 新手配置包。
- `ai-task-prompt-templates`：AI 任务提示词模板包。
- `ai-agent-safety-checklist`：AI Agent 权限与安全检查包。
- `static-site-starter`：个人静态网站起步模板。

## 使用说明

- 每个目录都包含 README 和可编辑文件。
- 下载 ZIP 后先阅读包内 README，再按自己的项目调整。
- 资料主要面向刚开始学习建站、编程、AI 工具和开源项目发布的新手。
- 内容会随网站文章和工具变化持续更新。
