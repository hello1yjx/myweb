# VS Code 新手配置包

适合希望减少格式差异、保存时自动整理文件，并建立稳定编辑习惯的新手。

## 包含文件

- `settings.json`：保守、通用的工作区配置
- `extensions.md`：按用途选择扩展的建议
- `starter.code-snippets`：HTML、JavaScript 和 Markdown 常用片段

## 使用方式

1. 在项目根目录创建 `.vscode` 文件夹。
2. 将 `settings.json` 和 `starter.code-snippets` 放入 `.vscode`。
3. 先确认项目已有格式化工具，再决定是否启用保存时格式化。
4. 不要一次安装大量扩展，先解决真实需求。
