# 个人静态网站起步模板

这是一个不依赖框架的 HTML、CSS、JavaScript 小网站模板，适合第一次练习个人主页或作品集。

## 文件

- `index.html`：页面结构
- `styles.css`：响应式样式
- `app.js`：自动年份和菜单交互
- `launch-notes.md`：修改与发布清单

## 本地使用

直接双击打开 `index.html` 即可预览。为了更接近正式网站，建议使用本地 HTTP 服务。

## 发布前必须修改

- 网站名称、简介和联系方式
- 项目卡片与真实链接
- 页面 title 和 description
- 不需要的示例内容
