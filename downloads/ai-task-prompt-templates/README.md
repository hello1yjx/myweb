# AI 任务提示词模板包

适合使用 ChatGPT、Codex、Copilot、Claude 等工具完成学习、写作、编程和资料整理任务的新手。

## 包含文件

- `prompt-templates.md`：6 类可复用任务模板
- `prompt-review-checklist.md`：提交前和结果复核清单
- `task-brief-template.md`：复杂任务说明模板

## 核心原则

好的提示词不是堆很多“高级词”，而是清楚说明：

1. 要完成什么结果
2. 已知背景和输入材料
3. 不能违反的限制
4. 怎样判断完成
5. 哪些地方必须人工确认
