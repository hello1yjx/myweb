# AI 提示词与结果复核清单

## 提交任务前

- [ ] 目标结果清楚，不只是“帮我优化”
- [ ] 已提供必要背景和真实输入
- [ ] 写清不能修改、不能泄露和不能执行的内容
- [ ] 指定输出格式和完成标准
- [ ] 敏感资料已删除或脱敏
- [ ] 涉及高风险操作时要求人工确认

## 收到结果后

- [ ] 事实、数字、日期和链接已核对
- [ ] 没有把推测当成事实
- [ ] 代码或命令理解后再执行
- [ ] 修改范围符合要求
- [ ] 已运行测试或实际打开结果验证
- [ ] 涉及账号、权限和费用的设置已人工检查
- [ ] 结果中没有泄露密钥或私人信息

## 需要停止并人工判断的情况

- AI 要求关闭安全控制
- AI 建议执行不理解的删除或覆盖命令
- 来源无法核实
- 结果涉及医疗、法律、财务等高风险决定
- 任务会访问生产环境、客户数据或付费账户
