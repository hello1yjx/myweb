# Git 与 GitHub 中文速查包

适合刚开始使用版本控制、经常忘记命令或担心误操作的新手。

## 包含文件

- `git-github-cheatsheet.md`：常用命令、日常流程与恢复思路
- `commit-message-template.txt`：提交说明模板
- `gitignore-starter.txt`：常见本地文件忽略模板

## 使用原则

- 提交前先看 `git status` 和 `git diff`
- 不确定时先保存现场，不使用破坏性命令
- 推送前确认分支和远端
- 重要改动写清楚验证结果
