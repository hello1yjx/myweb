# 静态网站上线检查表

适合第一次发布个人主页、作品集、博客和资料站的新手。

## 包含文件

- `website-launch-checklist.md`：上线前、上线后和定期维护检查项
- `page-content-inventory.csv`：页面内容与负责人清单
- `release-record-template.md`：每次发布的简短记录模板

## 建议用法

1. 将检查表复制到自己的项目仓库。
2. 每次上线前逐项确认，不适用的项目写明原因。
3. 发布后使用无痕窗口和手机检查用户真正看到的页面。
4. 记录发布日期、提交版本、已知问题和回滚方式。
