Google 回顾 A2A 一周年：多 Agent 协作开始从概念走向工程交接

Google Developers Blog 介绍 A2A 协议一周年进展，强调专门 Agent 可以通过共同协议安全接手任务，帮助主 Agent 避免上下文污染，并把生命科学、支付、实时数据、DevOps 和受监管网络等场景拆给专业能力处理。

代码审查 2026年6月18日

Copilot code review 支持仓库根目录 AGENTS.md，并改进草稿 PR 请求入口

GitHub Changelog 说明 Copilot code review 现在会读取仓库根目录的 AGENTS.md，并把相关说明用于审查反馈；同时草稿 PR 可以更直接地通过 reviewer picker 请求 Copilot，时间线中的部分 Copilot review 事件也会折叠显示。

为什么值得关注

这让团队约定、代码风格和审查重点更容易进入 AI 代码审查流程。新手应先把 AGENTS.md 写成可执行规则，而不是口号，并用草稿 PR 检查 Copilot 是否真的按仓库约定给出反馈。

候选：AGENTS.md 怎样写，才能让 AI 代码审查更贴近仓库约定
