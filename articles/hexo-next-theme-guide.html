Hexo / NexT 博客主题完整配置指南：官方文档、主题设置与扩展入口

2026年5月27日 9 分钟 作者：Avery Lin

给博客爱好者准备的 Hexo 与 NexT 主题资源页，覆盖安装、配置、常用功能和后续美化方向。

来源与增量价值

本文基于官方资料整理，并补充适用场景、限制条件和实践检查步骤。

首要来源：Hexo 官方文档。事实以原始来源为准，本站建议需结合自己的环境验证。

如果你想做博客，但又不想从头写前端，Hexo 配合 NexT 仍然是很适合中文用户的经典组合。资料多、生态成熟、配置项丰富，非常适合做长期内容站。

第一阶段我们先把它整理成资源导航页，后面再扩成完整图文教程和视频演示脚本。

这篇内容适合谁

  • 喜欢折腾博客样式和结构的个人站长
  • 已经接触过 Markdown 和命令行的博客爱好者

实践与验证步骤

  • 先看 Hexo 官方文档，理解初始化、写作和部署命令
  • 再看 NexT 文档，完成菜单、评论、代码高亮、SEO 等设置
  • 最后按自己的站点需求补评论系统、统计和主题美化

官方资料入口

  • Hexo 官方文档 ：安装、写作、部署和配置都以这里为准。
  • NexT 官方文档 ：主题菜单、侧边栏、评论和外观设置入口。
  • Hexo 配置说明 ：适合搭配主题配置一起看。

重点判断与使用建议

  • 优先整理最常用的 5 类配置：站点信息、菜单导航、评论系统、SEO、页面美化。
  • 录 GIF 或短视频时，重点演示改配置前后页面的变化，读者更容易理解。
  • 如果未来准备做付费主题或模板下载，这篇底部非常适合承接联盟或付费资源入口。

更适合落地的配置顺序

  • 先完成基础站点初始化，确保 Hexo 本地能正常启动和生成页面。
  • 再切到 NexT 主题，先改站点标题、菜单、头像和社交链接。
  • 然后补评论、代码高亮、搜索、SEO 和统计，不要一开始就改太多视觉细节。
  • 最后再做主题美化、页面动效和第三方服务接入。

适合加入正文的官方入口

  • Hexo Commands ：适合讲本地预览、生成和部署命令。
  • Hexo Writing ：适合解释文章、草稿和资源文件夹。
  • NexT Getting Started ：适合快速入门主题配置。

示例命令：本地启动与生成静态文件

如果这篇后面扩成长文教程，建议在最前面就放这组最常用命令。

hexo init my-blog
cd my-blog
npm install
hexo server
hexo generate
